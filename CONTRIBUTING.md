# Contributing to NodusAI Public Documentation

Thank you for your interest in contributing!

## What You Can Contribute

- Documentation improvements
- Example code and client libraries
- Bug fixes in examples
- Additional language examples

## What You Cannot Contribute

This repository does not accept contributions to:

- Core oracle logic (not included in this repo)
- Proprietary algorithms
- Production infrastructure

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/improvement`)
3. Make your changes
4. Test any code examples
5. Submit a pull request

## Code Style

- Use clear, descriptive variable names
- Include comments for complex logic
- Follow existing patterns in the codebase
- All examples must use mock endpoints (`https://api.example.com`)

## Documentation Style

- Use clear, concise language
- Include code examples where helpful
- Keep security in mind - never include real credentials

## Questions?

Open an issue for any questions about contributing.
