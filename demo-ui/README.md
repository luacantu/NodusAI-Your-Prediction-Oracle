# NodusAI Demo UI

A minimal, self-contained demonstration of the NodusAI assessment interface.

## Important Notice

**This demo uses MOCK DATA only.**

- No real API calls are made
- All responses are generated locally with random values
- This is for demonstration and UI preview purposes only

## Usage

Simply open `index.html` in any modern web browser:

```bash
open index.html

```
No build step, server, or dependencies required.

Features: 
- Query Input
- Source Selection
- Mock Response Generation
- JSON Viewer
- Dark Mode (automatic)
- Responsive Design

What This Does NOT Include: 

- Real API integration
- Actual likelihood calculations
- Proprietary oracle logic
- Production endpoints
